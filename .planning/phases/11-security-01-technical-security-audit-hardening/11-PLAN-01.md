## Objective
Mask all UIDs in Cloud Function logs to 8-char prefix + ellipsis, and completely remove email addresses from log output.

  <action>
    Find every console.log statement in functions/src/modules/auth/auth.functions.ts that interpolates a raw uid variable.
    Replace each bare `${uid}` inside console.log strings with `${uid.substring(0, 8)}...`.
